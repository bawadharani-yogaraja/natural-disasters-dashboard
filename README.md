# 🌍 Global Natural Disasters Dashboard

An interactive Streamlit dashboard analysing global natural disaster data from EM-DAT (2000–2026).

**Dataset:** EM-DAT – Aggregated figures for Natural Disasters
**Source:** Humanitarian Data Exchange (HDX)
**Module:** 5DATA004C Data Science Project Lifecycle – University of Westminster

## Features
- World map of disaster impact by country
- Disaster frequency trends over time
- Economic losses analysis (inflation-adjusted)
- Disaster type distribution
- Top 10 most affected countries
- Interactive filters for year, type, country

## How to Run
pip install -r requirements.txt
streamlit run app.py
